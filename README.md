# Support Desk App

Support ticket application built with the MERN stack. This is part of the React Front To Back 2022 Udemy course by Brad Traversy.

# Packages used:

- mongoose
- express
- axios
- react-icons
- react-modal
- reduxjs/toolkit
- react-toastify
